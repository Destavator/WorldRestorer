
	- Liste des commandes :
		/wr save <world> [as]									: Sauvegarde un monde. Poser la question pour écraser si la sauvegarde existe dejà
		/wr load <saved world> [as]								: Recharge un monde. Demander une confirmation
		/wr delete <saved world>								: Supprime une sauvegarde de monde. Demander une confirmation
		/wr list												: Liste des mondes suavegardés
		/wr config <saved world> setspawn						: Définit le spawn ou les joueurs seront téléporté lors du rechargement de ce monde.
		/wr config <saved world> kickplayers					: Indique que les joueurs doivent être kickés du serveur lors du rechargement de ce monde.
		/wr config <saved world> delay <seconds> [yyy]			: Définit un délai avant le rechargement du monde
		/wr config <saved world> command add <command>			: Ajout d'une commande qui sera executée après le rechargement du monde
		/wr config <saved world> command remove <command>		: Suppression d'une commande
		/wr config <saved world> command removeall				: Suppression de toutes les commandes
		/wr config <saved world> command list					: Liste des commandes
		/wr help
		/wr reload
